## üé® Brand Guidelines

### Logo Usage
- **Full Logo**: Use for general branding, marketing materials, and documentation
- **Logomark**: Use for app icons, favicons, and small spaces where text is not readable
- **White/Light versions**: Use on dark backgrounds
- **Dark versions**: Use on light backgrounds
- **Monochrome**: Use when color is not available or for special contexts

### File Format Guidelines
- **SVG**: Preferred for web, scalable graphics, and print (infinite scaling)
- **PNG**: Use for digital applications, social media, and when transparency is needed
- **ICO**: Use specifically for browser favicons

### Best Practices
- Maintain aspect ratios when resizing
- Provide adequate clear space around logos (minimum 10% of logo height)
- Do not distort, rotate, or modify logo colors
- Use high-resolution files for print materials
